# 🎉 AADHI Events & Entertainments

AADHI Events & Entertainments is a modern and responsive event management website designed to showcase professional event services, completed events, galleries, and customer booking features. The website provides an elegant user interface with smooth navigation and is built using Django for the backend.

---

## 🚀 Features

- Responsive Design
- Professional Landing Page
- Event Gallery
- Completed Events Showcase
- Event Categories
- About Us Section
- Contact Form
- Booking System
- Razorpay Payment Integration
- Image Gallery
- Smooth Animations
- Mobile Friendly UI

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AADHI-website.git
```

### Move into Project

```bash
cd AADHI-website
```

### Create Virtual Environment

```bash
python -m venv myenv
```

### Activate Virtual Environment

Windows

```bash
myenv\Scripts\activate
```

Linux/Mac

```bash
source myenv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000
```
